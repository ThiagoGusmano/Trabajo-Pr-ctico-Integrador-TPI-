El codigo cuenta con 6 funciones del sistema y una ultima para cerrar el programa
Estas funciones son: "Agregar un país" (Solo funciona para agregar un pais con todos sus datos de uno en uno), "Actualizar población y superficie" (Actualiza la poblacion y superficie del pais a eleccion), "Buscar un país" (Hace un ingreso exacto o parcial para detectar paises coincidentes, al detectarlos muestra los datos), "Filtrar países" (Esta funcion cuenta con 3 opciones, se puede filtrar por continente, rango de poblacion o rango de superficie y luego a traves de una funcion llamada imprimir tabla se embellece el resultado), "Ordenar países" (Esta funcion tambien cuenta con 3 opciones y estas cuentan con dos subopciones, las 3 primeras son: filtrar por nombre (alfabeticamente), por poblacion o por superficie, todas pueden ser elegidas en orden descendente o ascendente), "Mostrar estadísticas" (Esta funcion muestra el pais con mayor poblacion, el pais con menor poblacion, el promedio de poblacion mundial, el promedio de superficie global y la cantidad de paises por continente)
